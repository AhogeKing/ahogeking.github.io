                                                  输入格式

                                                  输入三个正整数，表示三条边的长度。

                                                  输出格式

                                                  如果能组成三角形，输出对应的三角形类型; 否则，输出“No!!!”

                                                  输入输出样例

                                                  提示

                                                  本题目涉及的三角类型有且只有”等边“、”等腰“、”一般”、“No!!!“。

                                                   

                                                  第四题 红莲斗篷

                                                  问题背景

                                                  在某游戏中，装备“红莲斗篷”（一件会灼烧附近事物的衣服）会对敌人造成基于英雄生命值的伤害。最近，游戏策划对红莲斗篷的伤害进行了调整，调整前的伤害为“50 + 3% 的额外生命值”，调整后变为“1.5% 的最大生命值”。额外生命值是指装备提供的附加生命值，而最大生命值是英雄的生命值和装备提供的额外生命值的总和。英雄的生命值随着等级提升，每升级都会增加一定的生命值，称为生命成长值。

                                                  请你编写一个程序，帮助学长判断同一英雄同一出装下的红莲斗篷效果是削弱了还是加强了。

                                                  问题描述

                                                  输入三个正数：英雄的初始最大生命值、生命成长值（每级增加的生命值）、和额外生命值（装备提供的附加生命值）。

                                                  调整后的伤害值在某一等级（1至满级之间）开始高于调整前时，输出“加强”及该等级，否则就输出“削弱”。满级为15级；一级时的生命值即为初始最大生命值，不需要计算生命成长值

                                                  输入格式

                                                  仅有一行，包括三个正数，中间用空格隔开。0⩽三个正数<107

                                                  输出格式

                                                  加强或者削弱，若是削弱，中间用空格隔开再输出一个正整数表示相应等级。

                                                  输入输出样例

                                                  提示

                                                  若在某个等级调整前的伤害首次低于调整后的伤害，那么在此等级之后的所有等级中，调整前的伤害将始终低于调整后的伤害。

                                                  第六题 完美找零计算器

                                                  问题描述

                                                  在一个遥远的星球上，居民们使用完美数的真因子作为纸币的面额。完美数是指一个等于其所有真因子之和的正整数。例如：

                                                  • 6是一个完美数，因为其真因子 1 + 2 + 3 = 6，所以他们的纸币面额为1元、2元和3元。

                                                  • 28是一个完美数，因为其真因子 1 + 2 + 4 + 7 + 14 = 28，因此纸币面额为1元、2元、4元、7元和14元。

                                                  现规定居民们在购物时可以使用任意两张不重复的纸币组合来支付商品的价格 P 元。

                                                  • 如果存在一组纸币组合，其总金额恰好等于P元，则输出该组合和0（表示没有找零）。

                                                  • 如果没有组合能恰好支付P元，但有组合的总金额大于P元，则选择总金额最接近且大于P元的组合，输出该组合和需要找零的金额

                                                  • 如果所有组合的总金额都小于P元，则选择总金额最接近且小于P元的组合，输出该组合和还差多少金额。

                                                  注意：

                                                  1. 只能选择两张面额不同的纸币进行组合

                                                  2. 每种面额的纸币只能使用一张

                                                  3. 保证不存在多个组合相加与price相等

                                                  输入格式

                                                  第一行输入两个正整数 n 和 P (n > 5, P > 0)，其中 n 为完美数，P 为商品的价格。中间用空格隔开。

                                                  输出格式

                                                  第一行输出两张纸币的面额，第二行输出需要找零的金额或者还差多少金额。
